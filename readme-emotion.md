# Emotion AI Detection System

## Overview
Emotion AI Detection System is a Machine Learning and Computer Vision project that detects human emotions from facial expressions in real time.

This project uses Deep Learning and Convolutional Neural Networks (CNNs) for facial emotion recognition through webcam or image input.

---

## Features
- Real-time emotion detection using webcam
- Facial expression recognition using AI/ML
- CNN-based deep learning model
- Image preprocessing and feature extraction
- Emotion classification with trained dataset
- Flask-based web interface

---

## Technologies Used
- Python
- TensorFlow / Keras
- OpenCV
- Flask
- NumPy
- Pandas

## Installation

### Clone the Repository
```bash
git clone <repository-link>
cd emotion-ai
```

### Install Dependencies
```bash
pip install tensorflow keras opencv-python flask numpy pandas
```

### Run the Application
```bash
python app.py
```

## How It Works
1. Captures webcam video.
2. Detects face using OpenCV.
3. Processes image using CNN.
4. Predicts emotion.
5. Displays detected emotion.
